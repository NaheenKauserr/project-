# Zero Click Analytics

Zero Click Analytics is an intelligent, automated dashboard. Upload any CSV or Excel file, and it automatically cleans the data, runs statistical analyses, creates dynamic visuals, generates KPIs, checks basic machine learning modeling/forecasting opportunities, and provides Gemini AI-powered insights and a Q&A chatbot over your datasets.

## Requirements & Setup

1. Install prerequisites:
   ```bash
   pip install -r requirements.txt
   ```

2. Add your Gemini API Key in the `.env` file!
   ```env
   GEMINI_API_KEY=your_actual_key
   ```

3. Run the app:
   ```bash
   streamlit run app.py
   ```
